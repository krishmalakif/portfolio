# Add Your Resume Here

Place your CV/Resume file in this folder with the name:

**`resume.pdf`**

The download button in the Hero section will link to:
`/portfolio-website/resume.pdf`

## Steps:
1. Save your resume as `resume.pdf`
2. Place it in the `public/` folder (same location as this file)
3. The file will be accessible at the root of your deployed site
