# Portfolio Website

Personal portfolio showcasing full-stack development projects and skills.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Export to PDF

Install browser (one-time):

```bash
npm run export:install
```

Generate PDF:

```bash
npm run export
```

Output: `submission/portfolio.pdf`

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Material UI

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run export` - Generate portfolio PDF
- `npm run export:install` - Install Playwright browser
